Welcome to StatMeal!

StatMeal is a custom graph and predictive model maker.
StatMeal was originally used for sorting customer receipt data from restaurants, but this program can be used to sort multiple kinds of data.

Getting Started

Select a spreadsheet file to use for the graph maker.
The file must be in a .csv format.
Make sure it the file is organized in columns with the data type listed at the head of the column.
